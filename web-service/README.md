# COE test web client
vue application for test frontend app
- 8080 PORT 사용

# getting started
```bash
$ git clone https://github.com/SDSACT/coe-cors-test-client.git
$ cd coe-cors-test-client
$ npm install
$ npm run dev
```

# Run with
- [Websocket server](https://github.com/SDSACT/coe-chat-service)
- 다른 REST API 서버
